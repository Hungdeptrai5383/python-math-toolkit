# Python Friendly Math Toolkit

A beginner level Python library demonstrating object-oriented
programming, operator overloading, and unit testing.

## Features
- Fraction simplification
- Arithmetic operations
- Fraction comparison
- Safe division handling

## Example
```python
from math_toolkit.fraction import Fraction

a = Fraction(2, 3)
b = Fraction(1, 2)

print(a + b)  # 5/6
print(a * b)  # 1/6
